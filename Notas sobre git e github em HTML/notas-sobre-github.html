Notas sobre o GitHub

Enquanto o Git é um software que pode ser instalado em sua máquina, o GitHub é uma plataforma online, em que há possibilidade de compartilhar os projetos criados em uma pasta local, sendo também uma ferramenta de divulgação de portfólio e projetos para os desenvolvedores. Há três versões da plataforma disponíveis:

  1. Web
  2. Aplicativo
  3. Desktop
