O que é Git?

É um command line interface (CLI). Em português, se traduz por "linha de comando". Esse software é bastante utilizado por desenvolvedores, pois além de servir para controlar e organizar projetos, funciona em diferentes sistemas operacionais. Em relação a segurança, não deixa a desejar. É possível acompanhar cada mudança feita dentro do repositório, podendo adicionar, clonar, alterar, etc.

Possibilidades de comandos no Git:

  • Mudar de pastas
  • Listar as pastas
  • Criar pastas e arquivos
  • Deletar pastas e arquivos
O programa está disponível para Windows, Linux e MacOS.
